# TicTacToe
This Java file defines the game board logic for a console-based Tic Tac Toe game.

Features

Represents a 3x3 Tic Tac Toe board.

Tracks the current player (X or O).

Validates moves and throws an InvalidMoveException for invalid ones.

Detects when the game ends (moveCount == 9).

Provides a formatted string representation of the board for console display.

Class Information

Class name: Board.java

Package: tictactoe

Usage

Use this class in a Tic Tac Toe project to manage the game board, check for valid moves, and handle player turns.
The toString() method can be used to display the board in the console after each move.
